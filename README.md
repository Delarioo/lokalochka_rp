# lokalochka_rp
Локалочка РП - это мой проект, разработанный на базе HELIX, в котором я учился программировать на Lua на основе своих скудных знаний и умений. В сборке встречаются баги, ошибки и недочёты. Код во многих местах неоптимизирован, посему плагины режима не подходят для публичных серверов и нуждаются в переработке. 
Проект заброшен, так как развивать локалку я больше не желаю. 
Консультации по сборке не предоставляю, смотрите код и изучайте её работу самостоятельно. 
Распространяется по принципу "как есть". Ничего не имею против взаимствований итд. 
Скачать можно [отсюда](https://github.com/Delarioo/lokalochka_rp/releases/tag/sourse)
